# Главная страница

![mainpage.png](static/MainPage/Главная%20страница.png)

#### На главной странице можно
- Просмотреть объявления
- Перейти к конкретному объявлению с помощью клика
- Проверить возможность доставки товара или наличие безопасной сделки наведением на значок дополнительной опции
- Перейти к конкретному объявлению с помощью клика на значок дополнительной опции
- Произвести поисковой запрос с помощью кнопки enter
- Произвести поисковой запрос с помощью клика по элементу всплывающего списка
- Перейти на страницу размещения товара 
- Перейти на страницу входа в аккаунт _(только для **не**авторизованного пользователя)_
- Перейти на страницу регистрации _(только для **не**авторизованного пользователя)_
- Перейти на главную страницу сайта
- Перейти в профиль пользователя _(только для авторизованного пользователя)_
- Перейти в корзину _(только для авторизованного пользователя)_
- Выйти из профиля _(только для авторизованного пользователя)_

*Все пункты, кроме первых трех, относятся к навбару, в данном разделе будут рассмотрены только первые 3

#### Просмотр объявлений
При открытии страницы отображается ее "нулевое состояние". Товары подгружаются по 20 штук в случае достижения конца страницы
![blankpage.png](static/MainPage/Загрузка%20товаров.png)

#### Переход к конкретному объявлению при помощи клика
При клике на значок товара, открывается его страница

#### Проверка дополнительных параметров
Около карточек товаров присутствует отдельный элемент, сигнализирующий о дополнительных параметрах продажи. При наведении на такой значок всплывает подсказка с названием дополнительной опции

![deliverypage.png](static/MainPage/Возможна%20доставкаpng.png)
![trustedpage.png](static/MainPage/Безопасная%20сделка.png)

* Не у всех товаров с плашкой "возможна доставка" доставка возможна

![apartmentpage.png](static/MainPage/Квартира.png)

#### Переход к конкретному объявлению при помощи клика на значок дополнительной опции
При клике на значок дополнительной опции происходит переадресация на страницу товара

#### Произвести поисковой запрос с помощью кнопки enter
В поиске пользователь вводит свой запрос и при нажатии клавиши enter, товары на главной странице заменяются на те, которые удовлетворяют поисковому запросу

* Если быстро написать запрос в строку поиска и отправить его, то выведется последний удачный поисковой запрос (или главная страница сайта)
* Если поисковый запрос не имеет совпадений, то будет выведен последний удачный поисковый запрос

#### Произвести поисковой запрос с помощью клика по элементу всплывающего списка
В поиске пользователь вводит свой запрос и получает всплывающий список товаров, в названии которых присутствуют нужные буквы. При клике на подходящий элемент из всплывающего списка, товары на главной странице заменяются на те, которые удовлетворяют поисковому запросу (тело поискового запроса соответствует тексту выбранного элемента из всплывающего списка)

![searchpage.png](static/MainPage/Поиск.png)

* Всплывающий список невозможно убрать с экрана без изменения запроса в поисковой строке

#### Переход на страницу размещения товара
Если пользователь не авторизован, то при клике на кнопку "Разместить объявление" происходит переадресация на страницу входа в аккаунт

Если пользователь авторизован, то при клике на кнопку "Разместить объявление" происходит переадресация на страницу создания объявления о продаже товара

#### Переход на страницу входа в аккаунт
Данная функция доступна только для неавторизованных пользователей. При нажатии на кнопку "Войти" происходит переадресация на страницу входа в аккаунт

#### Переход на страницу регистрации
Данная функция доступна только для неавторизованных пользователей. При нажатии на кнопку "Зарегистрироваться" происходит переадресация на страницу регистрации

#### Переход на главную страницу сайта
При нажатии на логотип происходит переадресация на главную страницу сайта

#### Переход в профиль пользователя
Данная функция доступна только для авторизованных пользователей. При нажатии на профиль в всплывающем окне можно выбрать пункт "Профиль" и происходит переадресация на профиль пользователя

#### Переход в корзину
Данная функция доступна только для авторизованных пользователей. При нажатии на значок корзины происходит переадресация на страницу корзины. Доступно только авторизированным пользователям

#### Выход из профиля
Данная функция доступна только для авторизованных пользователей. При нажатии на профиль в всплывающем окне можно выбрать пункт "Выйти" тогда пользователь становится неавторизованным. Доступно только авторизированным пользователям

# Корзина пользователя

![cart.png](static/ShoppingCart/Корзина%20полная.png)

#### В корзине можно
- Просмотреть список товаров
- Перейти на главную страницу (если товары отсутствуют в корзине)
- Перейти на страницу товара с помощью клика по товару
- Изменить количество товаров в корзине
- Удалить позицию из корзины
- Оформить заказ
- Видеть кол-во товаров в корзине и их стоимость

#### Просмотр списка товаров
В корзине отображаются товары, которые пользователь положил в корзину. Товары имеют кол-во, стоимость, описание и имя продавца

* При количестве товаров, превышающем 4, список товаров уходит за пределы фона страницы

![widecart.png](static/ShoppingCart/Корзина%20за%20границы.png)

#### Переход на главную страницу
При нажатии на кнопку перейти на главную страницу происходит переадресация на главную страницу сайта

![blankcart.png](static/ShoppingCart/пустая%20корзина.png)

#### Переход на страницу товара с помощью клика по картинке товара
При нажатии на картинку товара происходит переадресация на страницу данного товара

* При наведении на картинку товара, курсор остается неизменным, хотя пользователь может перейти на товар с помощью клика

#### Изменение кол-ва товаров в корзине
Если текущее количество товара меньше, чем доступное, то у товара будет доступна кнопка "+", при нажатии на которую кол-во товара в корзине увеличится на 1.
Если текущее количество товара больше, чем 1, то у товара будет доступна кнопка "-", при нажатии на которую кол-во товара в корзине уменьшится на 1.

*Позиция и размер кнопок, отвечающих за кол-во товара, зависят от ширины поля с названием товара

![zigzagcart.png](static/ShoppingCart/Поеховшая%20корзина.png)

#### Удаление позиции из корзины
При нажатии на значок мусорки, происходит полное удаление товара из корзины, независящее от количества единиц товара

#### Оформление заказа
При нажатии кнопки "Оформить заказ", все товары пропадают из корзины и становятся недоступными для покупки. Так же товары перемещаются во вкладку "Заказы" данного пользователя

#### Кол-во товаров в корзине и их стоимость
При наличии товаров в корзине, около значка корзины будет гореть кол-во товаров
Внутри корзины так же можно увидеть кол-во товаров и их общую стоимость
