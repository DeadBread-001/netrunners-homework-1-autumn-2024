## 3. Страница создания объявления

![create_product.png](static%2Fproduct_page%2Fcreate_product.png)

### 3.1 Поле названия

* При заполнении латиницей или цифрами от 1 до 256 символов успешно создается объявление

* Если поле оставить пустым, то будет высвечено сообщение "Вы пропустили это поле."

* При заполнении более чем 256 символами появляется ошибка "Ошибка валидации объявления: Заголовок должен быть длинной от 1 до 256 символов"

### 3.2 Вставка изображения 

* При нажатии на кнопку "Выбрать файлы" открывается окно просмотра файлов на компьютере, файлы недопустимых (все кроме jpg, jpeg, png) расширений при этом добавить невозможно

* При добавлении изображения справа от кнопки отображается название файла с расширением

![add_image.png](static%2Fproduct_page%2Fadd_image.png)


### 3.4 Поле выбора города

* Выбрав город из списка объявление успешно создается

* По умолчанию в поле всегда стоит Москва

### 3.5 Поле описания

* При заполнении латиницей или цифрами от 1 до 4000 символов успешно создается объявление

* Если поле оставить пустым, то будет высвечено сообщение "Вы пропустили это поле."

* При заполнении более чем 4000 символами появляется ошибка "Ошибка валидации объявления: Описание должно быть длинной от 1 до 4000 символов"

### 3.6 Поле выбора категории

* Выбрав категорию из списка объявление успешно создается

* По умолчанию в поле всегда стоит Недвижимость

### 3.7 Поле цены

* При вводе корректного числа (целое число от 1 до 1.000.000.000.000.000.000) создание объявления проходит успешно

* Если поле оставить пустым, то будет высвечено сообщение "Вы пропустили это поле."

* Если ввести сумму больше 1.000.000.000.000.000.000, появляется ошибка "Произошла внутренняя ошибка"

### 3.8 Поле доступного кол-ва товара

* При вводе корректного числа (целое число от 1 до 9999999999) создание объявления проходит успешно

* Если поле оставить пустым, то будет высвечено сообщение "Вы пропустили это поле."

* Если ввести сумму больше 9999999999, появляется ошибка "Произошла внутренняя ошибка"

#### 3.9 Баги страницы создания объявления

* Имеется возможность в поля с названием и описанием добавить zalgo текст

![zalgo_check.png](static/product_page/zalgo_check.png)

* В поле с названием и описанием можно вставить смайлики

![smile.png](static%2Fproduct_page%2Fsmile.png)

* При вводе 0 в поле "цена" и "доступное кол-во товара вылетает" не юзер френдли сообщение об ошибке

![create_product_error.png](static/product_page/create_product_error.png)

* При попытке прикрепить файл размером больше допустимого (1 МБ) не возникает никаких ошибок или предупреждений со стороны интерфейса

## 5. Главная страница

![mainpage.png](static/MainPage/Главная%20страница.png)

### 5.1 Просмотр объявлений
* При открытии страницы отображается ее "нулевое состояние". Товары подгружаются по 20 штук в случае достижения конца страницы

![blankpage.png](static/MainPage/Загрузка%20товаров.png)

### 5.2 Переход к конкретному объявлению при помощи клика
* При клике на значок товара, открывается его страница

### 5.3 Проверка дополнительных параметров
* Около карточек товаров присутствует отдельный элемент, сигнализирующий о дополнительных параметрах продажи. При наведении на такой значок всплывает подсказка с названием дополнительной опции

![deliverypage.png](static/MainPage/Возможна%20доставкаpng.png)
![trustedpage.png](static/MainPage/Безопасная%20сделка.png)

### 5.4 Переход к конкретному объявлению при помощи клика на значок дополнительной опции
* При клике на значок дополнительной опции происходит переадресация на страницу товара

### 5.5 Произвести поисковой запрос с помощью кнопки enter
* В поиске пользователь вводит свой запрос и при нажатии клавиши enter, товары на главной странице заменяются на те, которые удовлетворяют поисковому запросу

### 5.6 Произвести поисковой запрос с помощью клика по элементу всплывающего списка
* В поиске пользователь вводит свой запрос и получает всплывающий список товаров, в названии которых присутствуют нужные буквы. При клике на подходящий элемент из всплывающего списка, товары на главной странице заменяются на те, которые удовлетворяют поисковому запросу (тело поискового запроса соответствует тексту выбранного элемента из всплывающего списка)

![searchpage.png](static/MainPage/Поиск.png)

### 5.7 Переход на страницу размещения товара
* Если пользователь не авторизован, то при клике на кнопку "Разместить объявление" происходит переадресация на страницу входа в аккаунт

* Если пользователь авторизован, то при клике на кнопку "Разместить объявление" происходит переадресация на страницу создания объявления о продаже товара

### 5.8 Переход на страницу входа в аккаунт
* Данная функция доступна только для неавторизованных пользователей. При нажатии на кнопку "Войти" происходит переадресация на страницу входа в аккаунт

### 5.9 Переход на страницу регистрации
* Данная функция доступна только для неавторизованных пользователей. При нажатии на кнопку "Зарегистрироваться" происходит переадресация на страницу регистрации

### 5.10 Переход на главную страницу сайта
* При нажатии на логотип происходит переадресация на главную страницу сайта

### 5.11 Переход в профиль пользователя
* Данная функция доступна только для авторизованных пользователей. При нажатии на профиль во всплывающем окне можно выбрать пункт "Профиль" и происходит переадресация на профиль пользователя

### 5.12 Переход в корзину
* Данная функция доступна только для авторизованных пользователей. При нажатии на значок корзины происходит переадресация на страницу корзины. Доступно только авторизированным пользователям

### 5.13 Выход из профиля
* Данная функция доступна только для авторизованных пользователей. При нажатии на профиль во всплывающем окне можно выбрать пункт "Выйти" тогда пользователь становится неавторизованным. Доступно только авторизированным пользователям

### 5.14 Баги главной страницы

* Не у всех товаров с плашкой "возможна доставка" доставка возможна

![apartmentpage.png](static/MainPage/Квартира.png)

* Если быстро написать запрос в строку поиска и отправить его, то выведется последний удачный поисковой запрос (или главная страница сайта)

* Если поисковый запрос не имеет совпадений, то будет выведен последний удачный поисковый запрос

* Всплывающий список невозможно убрать с экрана без изменения запроса в поисковой строке

## 6. Корзина пользователя

![cart.png](static/ShoppingCart/Корзина%20полная.png)

### 6.1 Просмотр списка товаров
* В корзине отображаются товары, которые пользователь положил в корзину. Товары имеют кол-во, стоимость, описание и имя продавца

### 6.2 Переход на главную страницу
* При нажатии на кнопку перейти на главную страницу происходит переадресация на главную страницу сайта

![blankcart.png](static/ShoppingCart/пустая%20корзина.png)

### 6.3 Переход на страницу товара с помощью клика по картинке товара
* При нажатии на картинку товара происходит переадресация на страницу данного товара

### 6.4 Изменение кол-ва товаров в корзине
* Если текущее количество товара меньше, чем доступное, то у товара будет доступна кнопка "+", при нажатии на которую кол-во товара в корзине увеличится на 1.
* Если текущее количество товара больше, чем 1, то у товара будет доступна кнопка "-", при нажатии на которую кол-во товара в корзине уменьшится на 1.

### 6.5 Удаление позиции из корзины
* При нажатии на значок мусорки, происходит полное удаление товара из корзины, независящее от количества единиц товара

### 6.6 Оформление заказа
* При нажатии кнопки "Оформить заказ", все товары пропадают из корзины и становятся недоступными для покупки. Так же товары перемещаются во вкладку "Заказы" данного пользователя

### 6.7 Кол-во товаров в корзине и их стоимость
* При наличии товаров в корзине, около значка корзины будет гореть кол-во товаров
* Внутри корзины так же можно увидеть кол-во товаров и их общую стоимость

### 6.8 Баги страницы пользователя
* При количестве товаров, превышающем 4, список товаров уходит за пределы фона страницы

![widecart.png](static/ShoppingCart/Корзина%20за%20границы.png)

* При наведении на картинку товара, курсор остается неизменным, хотя пользователь может перейти на товар с помощью клика

* Позиция и размер кнопок, отвечающих за кол-во товара, зависят от ширины поля с названием товара

![zigzagcart.png](static/ShoppingCart/Поеховшая%20корзина.png)

* Если название товаров состоит более, чем из 20 символов, то оно выезжает за пределы карточки товара

![buyed_long_name](/static/sapranov/buyed_long_name.jpg)
![sold_long_name](/static/sapranov/sold_long_name.jpg)

* Если заказанные или проданные товары занимают больше трех строк, то нижние объявления принудительно сдвигают границы сайта

![too_many_orders](/static/sapranov/too_many_orders.jpg)

* Если прокрутить вниз, то пропадает header с поиском, корзиной и профилем. Чтобы воспользоваться данными функциями, приходится возвращаться в самый верх сайта

![no_header](/static/sapranov/no_header.jpg)
